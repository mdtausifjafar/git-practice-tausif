## What This Project Does
This project demonstrates the use of Git and GitHub. It includes a basic calculator program built in Python, showing how to create repositories, make commits, work with branches, and push code to GitHub.

## Project Structure

- `src/main.py` - Main program file
- `src/utils.py` - Utility/calculator functions
- `docs/project-description.md` - Project documentation
